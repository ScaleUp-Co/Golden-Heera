# Golden Heera Luxury Boutique

A stunning high-fashion minimalist-luxury boutique e-commerce web platform for Golden Heera, crafted where traditional Indian craftsmanship meets contemporary digital precision.

## Technology Stack
- **HTML5** & semantic structures
- **Tailwind CSS** (V3 via CDN)
- **Google Fonts** (Montserrat & Playfair Display)
- **Material Symbols Outlined** (Icons)
- **JavaScript (Vanilla ES6)** for client-side filtering, state tracking, carousel controls, interactive zoom and checkout validation.

## Pages Included
- **Home (`index.html`)**: Features elegant branding, bento grid showcases, curations, and responsive design.
- **Collections (`collections.html`)**: Dynamic filtering by type, material, price range, and sorting.
- **Product Detail (`product-detail.html`)**: Detailed view of "The Noor Mahal Necklace" featuring certificate check, zoom magnification, and add-to-cart logic.
- **Shopping Cart (`cart.html`)**: Real-time quantity adjustments, promo codes (try `GIFT10`), and tax/shipping calculations.
- **Checkout (`checkout.html`)**: Responsive accordion step-by-step progress billing, shipping inputs, Razorpay/credit-card tab switches, and order placement success screen.
